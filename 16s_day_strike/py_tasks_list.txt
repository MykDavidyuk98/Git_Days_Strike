1. Виведення чисел, пропускаючи певні значення
   Умова: Вивести числа від 1 до 100, пропускаючи кратні 5.
   Тип: Цикл з continue.

2. Виведення парних чисел у діапазоні
   Умова: Вивести всі парні числа від -20 до 20, пропускаючи нуль.
   Тип: Цикл + continue + умова.

3. Виведення букв з рядка, окрім голосних
   Умова: Задано рядок. Вивести всі символи, окрім голосних літер.
   Тип: Цикл for по рядку + continue.

4. Виведення чисел від 10 до 1, крім чисел 3 і 7
   Умова: Зворотній відлік від 10 до 1, пропускаючи 3 і 7.
   Тип: Цикл з кроком -1 + continue.

5. Ввід цілого числа з перевіркою
   Умова: Користувач має ввести ціле число. Якщо ввів не число — вивести повідомлення про помилку і запросити ще раз.
   Тип: Перевірка типу (int).

6. Ділення з захистом від нуля
   Умова: Запросити два числа a і b. Вивести результат a / b. Якщо b == 0 — вивести повідомлення про помилку, запросити b знову.
   Тип: Перевірка на ділення на нуль.

7. Ввід додатного числа
   Умова: Запросити у користувача додатне число (> 0). Якщо ввів 0, від’ємне або не число — повторити ввід.
   Тип: Перевірка значення + типу.

8. Введення натурального числа
   Умова: Користувач має ввести натуральне число. Якщо ввід некоректний (від’ємне або 0) — запит повторюється.
   Тип: Перевірка значення.

9. Вибір з меню
   Умова: Користувач обирає одну з опцій меню: 1 — Сума, 2 — Різниця, 3 — Добуток. Якщо обрано щось інше — вивести помилку й повторити запит.
   Тип: Вибір із обмеженого набору.

10. Введення дробового числа
    Умова: Запитати у користувача дробове число (float). Якщо ввід некоректний — просити ще раз.
    Тип: Перевірка формату.

11. Факторіал числа (рекурсія)
    Умова: Обчислити факторіал числа N рекурсивним способом.
    Вхід: Ціле число N ≥ 0.

12. Сума цифр числа
    Умова: Знайти суму цифр натурального числа N за допомогою рекурсії.
    Вхід: Додатне ціле число.

13. Піднесення до степеня
    Умова: Піднести число a до натурального степеня n рекурсивно.
    Вхід: Дійсне a, ціле n ≥ 0.
